### 案例說明1
|   使用案例分析   |  登入系統   |
| :--------|:-------|
| 行動者  | 顧客  |
| 說明 |  輸入帳號密碼登入系統 |
| 完成動作 | 1. 顧客輸入帳號<br>2. 顧客輸入密碼<br>3. 顧客點擊登入<br>4. 系統顯示成功登入 |
| 替代動作 | 1. 顧客輸入帳號<br>2. 顧客輸入密碼<br>3. 顧客點擊登入<br>4. 系統顯示無法登入 |
| 先決條件  |  帳號與密碼不可空白 |
| 後置條件 | 登入成功才可選購商品 |
| 假設 | 無 |
